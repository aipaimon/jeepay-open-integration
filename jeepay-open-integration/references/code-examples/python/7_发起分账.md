# 发起订单分账

API 路径：`api/division/exec`

SDK 调用：`jeepay.Division.exec(**params)`

> 请求/响应参数详见 [发起订单分账文档](https://doc.jeequan.com/#/integrate/open/api/97)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"channelBatchOrderId":"4200001234202406181234567890"}}
```

**失败响应**：

```json
{"code":9999,"msg":"订单未支付成功，不允许分账"}
```

## 完整代码示例

```python
import json
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        # 方式一：使用系统自动分账组
        # result = jeepay.Division.exec(
        #     payOrderId="P1470667876906389505",          # 支付订单号
        #     useSysAutoDivisionReceivers=1,               # 使用系统自动分账组
        # )

        # 方式二：手动指定接收者
        receivers = json.dumps([
            {"receiverId": "800004", "receiverGroupId": "", "divisionProfit": "0.1"},
            {"receiverId": "800005", "receiverGroupId": "", "divisionProfit": "0.2"},
        ])

        result = jeepay.Division.exec(
            payOrderId="P1470667876906389505",          # 支付订单号
            # mchOrderNo="mho1718712345678",            # 商户订单号（与 payOrderId 二选一）
            useSysAutoDivisionReceivers=0,               # 手动指定接收者
            receivers=receivers,                         # 接收者列表 JSON
        )

        log.info("[Jeepay] 分账成功: %s", result)
        log.info("渠道分账订单号: %s", result.get("channelBatchOrderId"))

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 分账失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 分账异常: %s", e)
```
