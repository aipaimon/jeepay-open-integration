# 绑定分账用户

API 路径：`api/division/receiver/bind`

SDK 调用：`jeepay.Division.bind_user(**params)`

> 请求/响应参数详见 [绑定分账用户文档](https://doc.jeequan.com/#/integrate/open/api/96)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"receiverId":"800004","accNo":"oMock123456","accName":"测试商户","bindState":1}}
```

**失败响应**：

```json
{"code":9999,"msg":"接收账号已绑定"}
```

## 完整代码示例

```python
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Division.bind_user(
            ifCode="wxpay",                           # 支付接口代码
            accType=1,                                 # 1-商户(对公)
            accNo="接收账号",                            # 如 openid
            accName="接收人名称",                        # 账号持有人名称
            relationType="SERVICE_PROVIDER",           # 关系类型
            relationTypeName="服务商",                   # 关系类型名称
            divisionProfit="0.10",                     # 分账比例 10%
            # receiverAlias="",                         # 接收者别名
            # receiverGroupId="",                       # 分账接收者组ID
        )

        log.info("[Jeepay] 绑定分账用户成功: %s", result)
        log.info("accNo: %s, 绑定成功", result.get("accNo"))

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 绑定分账用户失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 绑定分账用户异常: %s", e)
```
