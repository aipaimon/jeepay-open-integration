# 转账

API 路径：`api/transferOrder`

SDK 调用：`jeepay.Transfer.create(**params)`

> 请求/响应参数详见 [发起转账订单文档](https://doc.jeequan.com/#/integrate/open/api/93)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"transferId":"T202406181543441860003","mchOrderNo":"tfo1718712345678","state":1}}
```

**失败响应**：

```json
{"code":9999,"msg":"收款账号不存在"}
```

## 完整代码示例

```python
import time
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Transfer.create(
            mchOrderNo=f"tfo{round(time.time() * 1000)}",  # 商户转账单号
            amount=1,                                        # 转账金额（单位：分）
            currency="CNY",                                  # 货币代码，固定CNY
            ifCode="wxpay",                                  # 微信转账
            entryType="WX_CASH",                             # 微信零钱
            accountNo="用户openid",                           # 收款人 openid
            accountName="收款人姓名",                          # 收款人姓名
            transferDesc="测试转账",                           # 转账备注
            clientIp="192.168.1.100",                        # 客户端IP地址
            notifyUrl="https://你的域名/notify/transfer",      # 转账异步通知地址
        )

        log.info("[Jeepay] 转账成功: %s", result)
        log.info("transferId: %s", result.get("transferId"))
        log.info("mchOrderNo: %s", result.get("mchOrderNo"))

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 转账失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 转账异常: %s", e)
```
