# 查询转账

API 路径：`api/transfer/query`

SDK 调用：`jeepay.Transfer.transfer_query(**params)`

> 请求/响应参数详见 [查询转账订单文档](https://doc.jeequan.com/#/integrate/open/api/94)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"transferId":"T202406181543441860003","mchOrderNo":"tfo1718712345678","appId":"60cc09bce4b0f1c0b83761c9","mchNo":"M1621873433953","amount":1,"currency":"CNY","state":2,"accountNo":"oMock123456","accountName":"测试用户","successTime":"2024-06-18 15:44:00"}}
```

**失败响应**：

```json
{"code":9999,"msg":"转账订单不存在"}
```

## 完整代码示例

```python
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Transfer.transfer_query(
            transferId="T202108121543441860003",     # 转账订单号
            # mchOrderNo="tfo1718712345678",          # 商户转账单号（与 transferId 二选一）
        )

        log.info("[Jeepay] 查询转账成功: %s", result)
        log.info("转账状态: %s", result.get("state"))
        log.info("转账金额: %s", result.get("amount"))

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 查询转账失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 查询转账异常: %s", e)
```
