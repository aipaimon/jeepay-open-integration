# 关闭订单

API 路径：`api/pay/close`

SDK 调用：`jeepay.Pay.pay_close(**params)`

> 请求/响应参数详见 [关闭支付订单文档](https://doc.jeequan.com/#/integrate/open/api/87)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS"}
```

**失败响应**：

```json
{"code":9999,"msg":"当前订单不可关闭"}
```

## 完整代码示例

```python
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Pay.pay_close(
            payOrderId="P1485879219030011906",     # 要关闭的支付订单号
            # mchOrderNo="mho1718712345678",        # 商户订单号（与 payOrderId 二选一）
        )

        log.info("[Jeepay] 关闭订单成功: %s", result)

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 关闭订单失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 关闭订单异常: %s", e)
```
