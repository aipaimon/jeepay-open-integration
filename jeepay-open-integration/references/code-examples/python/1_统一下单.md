# 统一下单

API 路径：`api/pay/unifiedOrder`

SDK 调用：`jeepay.Pay.create(**params)`

> 请求/响应参数详见 [统一支付下单文档](https://doc.jeequan.com/#/integrate/open/api/85)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"payOrderId":"P2056570316211871745","mchOrderNo":"mho1718712345678","orderState":1,"payDataType":"codeUrl","payData":"weixin://wxpay/bizpayurl?pr=abc123"}}
```

**失败响应**：

```json
{"code":9999,"msg":"订单已过期"}
```

## 完整代码示例

```python
import time
import json
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Pay.create(
            mchOrderNo=f"mho{round(time.time() * 1000)}",    # 商户订单号
            wayCode="WX_NATIVE",                               # 支付方式
            amount=1,                                          # 支付金额（单位：分）
            currency="cny",                                    # 货币代码
            clientIp="192.168.1.100",                          # 客户端IP地址
            subject="测试商品",                                  # 商品标题
            body="商品描述",                                    # 商品描述
            notifyUrl="https://你的域名/notify/pay",             # 异步通知地址
            returnUrl="https://你的域名/return",                 # 前端跳转地址
            # channelExtra=json.dumps({"payDataType": "codeImgUrl"}),  # 渠道扩展参数（JSON）
            # extParam="",                                     # 商户扩展参数
            # expiredTime="",                                  # 订单失效时间
            # divisionMode=0,                                  # 分账模式
        )

        log.info("[Jeepay] 统一下单成功: %s", result)

        pay_data_type = result.get("payDataType")
        pay_data = result.get("payData")

        log.info("payOrderId: %s", result.get("payOrderId"))
        log.info("payDataType: %s", pay_data_type)
        log.info("payData: %s", pay_data)

        # payDataType 处理：
        # "payUrl" → 重定向到支付页面
        # "form"  → 渲染HTML表单并自动提交
        # "codeUrl" → 用 payData 生成二维码
        # "codeImgUrl" → 直接展示二维码图片

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 统一下单失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 统一下单异常: %s", e)
```
