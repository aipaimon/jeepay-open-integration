# 查询退款

API 路径：`api/refund/query`

SDK 调用：`jeepay.Pay.refund_query(**params)`

> 请求/响应参数详见 [查询退款订单文档](https://doc.jeequan.com/#/integrate/open/api/91)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"refundOrderId":"R2056570316211871745","mchRefundNo":"REF1779159547217","payOrderId":"P2056570316211871745","appId":"60cc09bce4b0f1c0b83761c9","mchNo":"M1621873433953","refundAmount":100,"currency":"CNY","state":2,"channelOrderNo":"DEMOMOCK_T2056570316258009089","createdAt":"2024-06-18 11:05:52","successTime":"2024-06-18 11:05:53"}}
```

**失败响应**：

```json
{"code":9999,"msg":"退款订单不存在"}
```

## 完整代码示例

```python
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Pay.refund_query(
            refundOrderId="P202106181105527690009",    # 退款订单号
            # mchRefundNo="REF1779159547217",          # 商户退款单号（与 refundOrderId 二选一）
        )

        log.info("[Jeepay] 查询退款成功: %s", result)
        log.info("退款状态: %s", result.get("state"))
        log.info("退款金额: %s", result.get("refundAmount"))

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 查询退款失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 查询退款异常: %s", e)
```
