# 查询订单

API 路径：`api/pay/query`

SDK 调用：`jeepay.Pay.pay_query(**params)`

> 请求/响应参数详见 [查询支付订单文档](https://doc.jeequan.com/#/integrate/open/api/86)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"payOrderId":"P2056570316211871745","mchOrderNo":"mho1718712345678","appId":"60cc09bce4b0f1c0b83761c9","mchNo":"M1621873433953","wayCode":"WX_NATIVE","amount":1,"currency":"CNY","orderState":2,"clientIp":"192.168.1.100","subject":"测试商品","createdAt":"2024-06-18 11:04:17","successTime":"2024-06-18 11:05:30"}}
```

**失败响应**：

```json
{"code":9999,"msg":"订单不存在"}
```

## 完整代码示例

```python
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Pay.pay_query(
            payOrderId="P202106181104177050002",    # 支付订单号
            # mchOrderNo="mho1718712345678",        # 商户订单号（与 payOrderId 二选一）
        )

        log.info("[Jeepay] 查询订单成功: %s", result)
        log.info("金额: %s", result.get("amount"))

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 查询订单失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 查询订单异常: %s", e)
```
