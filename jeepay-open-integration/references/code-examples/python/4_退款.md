# 退款

API 路径：`api/refund/refundOrder`

SDK 调用：`jeepay.Pay.refund(**params)`

> 请求/响应参数详见 [发起支付退款文档](https://doc.jeequan.com/#/integrate/open/api/90)

---

## 响应示例

**成功响应**：

```json
{"code":0,"msg":"SUCCESS","data":{"channelOrderNo":"DEMOMOCK_T2056570316258009089","mchRefundNo":"REF1779159547217","payAmount":100,"refundAmount":100,"refundOrderId":"R2056570316211871745","state":2}}
```

**失败响应**：

```json
{"code":9999,"msg":"退款金额超出订单可退金额"}
```

## 完整代码示例

```python
import time
import logging
import jeepay
from jeepay.app_config import AppConfig

logging.basicConfig(level=logging.INFO)
log = logging.getLogger(__name__)


def init_config():
    """初始化 SDK 配置（静态全局配置，适用于单商户场景）"""
    AppConfig.set_mch_no("你的商户号")
    AppConfig.set_app_id("你的应用ID")
    AppConfig.set_api_key("你的API密钥")
    # 如需修改网关地址（默认 http://pay.d.jeepay.vip）：
    # from jeepay import app_config
    # app_config._params_config["api_base"] = "https://你的网关地址"


if __name__ == "__main__":
    init_config()

    try:
        result = jeepay.Pay.refund(
            payOrderId="P202106181104177050002",        # 原支付订单号
            # mchOrderNo="mho1718712345678",            # 商户支付单号（与 payOrderId 二选一）
            mchRefundNo=f"rfd{round(time.time() * 1000)}",  # 商户退款单号
            refundAmount=1,                               # 退款金额，单位分
            currency="CNY",                               # 货币代码，固定CNY
            # clientIp="192.168.1.100",                  # 客户端IP地址
            refundReason="用户申请退款",                    # 退款原因
            notifyUrl="https://你的域名/notify/refund",    # 退款异步通知地址
            # channelExtra="",                            # 渠道扩展参数
            # extParam="",                                # 商户扩展参数
        )

        log.info("[Jeepay] 退款成功: %s", result)
        log.info("refundOrderId: %s", result.get("refundOrderId"))
        log.info("mchRefundNo: %s", result.get("mchRefundNo"))

    except jeepay.error.APIError as e:
        log.error("[Jeepay] 退款失败: %s", e)
    except Exception as e:
        log.error("[Jeepay] 退款异常: %s", e)
```
